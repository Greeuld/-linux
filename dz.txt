## С помощью текстового редактора Vim создать файл с программой на Python, выводящей текст Hello, world!

sudo apt install vim

vim hello.py

*нажимаем i*

*вводим*
print("Hello World!")

*нажимаем esc*

*вводим*
:wq

## Запустить команду, определяющую число строк в файле.

cat hello.py | wc -l


## Создать еще один файл с командой на Python, выводящей текст Linear regression.

vim linear.py

*нажимаем i*

*вводим*
print("Linear regression")

*нажимаем esc*

*вводим*
:wq

## Объединить эти два файла с помощью команды cat.

cat hello.py linear.py > final.py

## Придумать три случая применения команды cat для работы с текстовыми файлами.

1. Просмотр/чтение текстового файла

cat filename.txt

2. Перенаправление ввода 

cat < filename.txt

3. Перенаправление вывода

cat > filename.txt
